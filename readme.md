## 🚀 快速開始（開發者模式）

### 1️⃣ 環境準備
```bash
# 確保你有這些工具
- Go 1.20+ 📦
- MySQL 8+ 🗄️
- 一顆熱愛coding的心 ❤️
```

### 2️⃣ 克隆專案
```bash
git clone https://github.com/Justinkuo1053/taipei-day-trip.git
cd taipei-day-trip
# 歡迎來到台北一日遊的世界！
```

### 3️⃣ 環境設定
```bash
cp .env.example .env
# 編輯 .env，填入你的資料庫連線資訊
# 別擔心，我們不會偷看你的密碼 😉
```

### 4️⃣ 啟動服務
```bash
go run cmd/main.go
# 看到 "Server is running" 就代表成功了！
```

### 5️⃣ 開始探索
- 打開瀏覽器
- 前往 `http://localhost:8080`
- 開始你的台北冒險之旅！
